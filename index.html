作品及社团实践

这些是我曾经做过的一些作品们。它们有的是社团项目，有的是学校课题，有的是为了来满足我探索的好奇心的 :)

密码沙拉

密码沙拉是我在ACM社团里制作的一款简单易懂的网络安全的迷你课程，现为洛杉矶CityLab网络安全课程的一部分。 这个作品的内容包含探索密码的历史，以及如今网络加密的方法，例如密码学中的caesar密码，atbash码，vigenere码，以及现代的简化的Base64以及AES加密方法。 在制作方面，我们前端利用了Figma, Reactjs, Bulma, 和Animejs来设计及编写交互式内容，后端利用了Firebase来进行密码的保存及分享。

基因树

为提升现有基因存储程序的效率，帮助研究人员更好地搜索及比较常见疾病和细菌的DNA，我利用面向对象的开放方式使用C++编写了一个可以编辑、存储、查找以及匹配基因组序列的基因库，并由此更快的识别出变异的细菌菌株。 运用此程序，用户可以添加新生物基因组，查找基因组，以及在基因库里鉴定出与某一基因高度匹配的其他基因组。 在架构方面，我利用了字典树 (Trie) 的数据结构，以减少空间和时间复杂度，使程序可以快速完成用户匹配或修改指令。

AR校园向导

为改造现访问者浏览学校的方式，提升浏览效率并增加游览乐趣，我们团队合并了现有的静态学校地图介绍以及学生导览员，创造了iOS系统的增强现实(Augmented Reality)导览软件。 通过本软件，用户只需打开相机扫描学校建筑物，一款运用3D模型创作的学校吉祥物就会出现在屏幕上进行周围环境的讲解，包括教学楼背景、奇闻逸事、冼手间及餐厅导引。 在本软件中，我使用Figma设计了用户界面及提升用户体验、并在Xcode上使用swift编写了登陆界面及环境讲解介绍。 此外我们还运用微软Azure的文字到语音转换系统，为介绍内容提供同步语音朗读 This project is an IOS App to improve visitors' experience when touring UCLA by using an augmented reality Bruin introducing the various places on campus. By working on this, I gained valuable experience working as a group, where we implemented page functionality in Swift, loaded area-based introduction text based on location detection. In addition, I designed user experience using Figma, and resolved size conflict for the app to be responsive on different devices.

超分辨率评估

这篇论文评估了三种不同的通过机器学习提升核磁共振影像图分辨率的算法。 其中包括：增强型深剩余网络(EDSR)、快速精确成像超分辨率(RAISR)， 以及宽激活深度剩余网络(WDSR)。我们用了LaTeX编写了论文。在文中， 我们利用平均绝对误差，均方误差，和峰值信噪比来评估算法，并通过调整不同标准化权重和批标准化(BN)来比较算法结果。
